# Security

## Reporting a vulnerability

If you believe you have found a security vulnerability, please report it responsibly:

- **Do not** open a public issue.
- Contact the project maintainers (e.g. via the repository’s issue tracker or contact option) with a description of the issue and steps to reproduce, if possible.

We will acknowledge your report and work with you to understand and address it.

## Supported versions

We provide security updates for the current major version. Upgrade to the latest release to receive security fixes.
